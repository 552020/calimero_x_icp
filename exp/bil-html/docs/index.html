What is IBIL Token?

IBIL (Internal BIL) is the platform's mining reward token:

  • Earned by miners for successfully mining blocks
  • Can be transferred between wallets
  • Can be converted to BIL tokens

How IBIL Transactions Work:

1. All IBIL transactions are first sent to the mempool

2. Transactions in the mempool are grouped into blocks

3. Miners compete to mine these blocks

4. Successful miners receive IBIL tokens as rewards
